This web app is built upon Ruby On Rails

Ruby version => 2.3.1
Rails version => 2.4.6

Users can sign up to the app. They can search for the stock, track it and add to their specific portfolio. There is also the friendship model that allows users to add friend each other. They can view what stocks their friend are tracking.

Live preview of the app can be seen here => <a href="stock-tracker-rabin.herokuapp.com">Stock tracker</a>
